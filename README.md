# C++ Programming Assignment – 3

## 📌 Description
This repository contains solutions to **Assignment 3 of C++ Programming**.  
The assignment is divided into **3 sections**, covering:

- Arrays using indexing and pointer notation  
- Dynamic memory allocation (runtime arrays)  
- Logical and real-world problem solving  

All programs are written in **C++ (.cpp)** and compiled using **g++ (GCC compiler)**.

---

# 🛠 Concepts Used
- Arrays (`[]`)
- Pointer notation
- Dynamic memory allocation (`new`, `delete`)
- Loops (`for`, `while`)
- Conditional statements (`if-else`, `switch`)
- Arithmetic operations
- Logical thinking

# 📂 SECTION – 1  
### (Fixed Size Arrays + Pointer Notation)

### Q1  
Scan an array of 5 integers and copy it into another array using `[]` notation.

### Q2  
Scan an array of 8 integers and print the sum using pointer notation.

### Q3  
Scan an array of 6 characters and count uppercase alphabets.

### Q4  
Store squares of array elements into another array and print both arrays.

### Q5  
Find the largest value in an array using pointer notation.

### Q6  
Copy array elements into another array in reverse order.

### Q7  
Declare and initialize an array and find the largest element using pointer notation.

### Q8  
Declare and initialize an array and copy it into another array in reverse order using `[]` notation.

# 📂 SECTION – 3  
### (Logical & Real-World Programs)

### Q1  
Calculate Simple Interest and suggest:  
- Buy equity if SI > 5000  
- Invest in mutual fund otherwise  

### Q2  
Calculate land cost:  
- Area > 1000 → ₹6000 per sq ft  
- Area ≤ 1000 → ₹5500 per sq ft  

### Q3  
Calculate Diwali bonus:  
- Salary > 15000 → 20% bonus  
- Salary ≤ 15000 → 25% bonus  

### Q4  
Perform arithmetic operations based on user choice (1–4).

### Q5  
Gender-based logic program:  
- Male → age → salary / birth year  
- Female → birth year → salary / expected salary
