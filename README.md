# C++ Programming Assignment – 3

## 📌 Description
This repository contains solutions to **Assignment 3 of C++ programming**.  
The assignment focuses on practicing concepts such as **arrays, pointer notation, runtime memory arrays, conditional statements, and logical problem solving**.

All programs are written in **C++ (.cpp)** and compiled using the **GCC C++ compiler (g++)**.

The assignment is divided into **three sections**.

# 📂 Assignment Structure

## SECTION – 1
Programs based on **arrays using fixed size** and **pointer notation**.

### Q1
Scan an array of **5 integers** and copy it into another array.  
(Using **array indexing `[]` method**)

### Q2
Scan an array of **8 integers** and print the **addition of all elements**.  
(Using **pointer notation**)

### Q3
Scan an array of **6 characters** and count how many of them are **uppercase alphabets**.  
(Using **array indexing `[]` method**)

### Q4
Declare and initialize an array of **6 integers**.  
Store the **square of all elements** into another array and print both arrays.  
(Using **pointer notation**)

### Q5
Declare and initialize an array of **8 integers** and find the **largest value** from the array.  
(Using **pointer notation**)

### Q6
Declare and initialize an array of **8 integers** and copy it into another array in **reverse order**.  
(Using **array indexing `[]` notation**)

# SECTION – 3
Programs based on **logical decision making and conditional statements**.

### Q1
Calculate **Simple Interest** from user input.  

- If **SI > 5000** → Suggest user to **buy equity of total amount (principal + SI)**  
- If **SI ≤ 5000** → Suggest user to **invest total amount in mutual fund**  

(Use **if-else**, **switch case not allowed**)

---

### Q2
Scan **length and width of land (in feet)** and calculate **area**.

Cost condition:

- If **area > 1000 sq ft** → Rate = **6000 Rs per sq ft**
- If **area ≤ 1000 sq ft** → Rate = **5500 Rs per sq ft**

Calculate **total cost of land**.

---

### Q3
Calculate **Diwali bonus for employee**.

Input: **Basic Salary**

Conditions:

- Salary > 15000 → **20% bonus**
- Salary ≤ 15000 → **25% bonus**

Print **Total Salary = Basic Salary + Bonus**.

---

### Q4
Scan **two numbers** and ask user to choose operation:

1 → Addition  
2 → Subtraction  
3 → Division  
4 → Multiplication  

Print result according to user choice.

---

### Q5
Scan **gender from user**.

If gender is **male**:

- Scan **age**
- If **age ≥ 25** → Scan **salary** and print age and salary
- If **age < 25** → Ask **birth year** and print it

If gender is **female**:

- Ask **birth year** and calculate **current age**
- If **age ≥ 25** → Ask **salary** and print it
- If **age < 25** → Ask **expected salary** and print it
